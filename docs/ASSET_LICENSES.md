# Asset Licences

Every visual, audio, font and icon asset in SYSTEM is recorded here with its
origin and licence. Nothing is added to the project without an entry.

## Originality policy

SYSTEM draws atmosphere from progression-fantasy system interfaces but must
remain **legally distinct**. Explicitly prohibited:

- Any character, illustration, logo, emblem, dialogue, music, sound effect, icon,
  font or interface layout from Solo Leveling, Minecraft, or any other protected
  work.
- Assets downloaded without verifying the licence.
- "Inspired by" reproductions close enough to be recognisable as the original.

Permitted: extracting high-level qualities — dark dimensional atmosphere, blue
and violet accents, layered translucent panels, choreographed reveals — and
expressing them through original work.

## Current inventory

### Audio — ✅ fully original

| Asset                             | Origin                                                                                 | Licence                             |
| --------------------------------- | -------------------------------------------------------------------------------------- | ----------------------------------- |
| All 10 interface and event sounds | **Procedurally synthesised at runtime** via the Web Audio API in `src/audio/engine.ts` | Original work, part of this project |

No audio files exist in the repository. Every sound is generated from oscillators
and gain envelopes at play time. This was chosen deliberately: it eliminates
licensing exposure entirely, adds zero install size, and keeps the sound identity
coherent because all pitches derive from one defined minor-pentatonic set.

### Fonts — ⚠️ not yet bundled

| Requested      | Fallback in use                               | Licence                   | Status         |
| -------------- | --------------------------------------------- | ------------------------- | -------------- |
| Inter          | Segoe UI Variable / Segoe UI (Windows system) | SIL Open Font License 1.1 | ⬜ not bundled |
| JetBrains Mono | Cascadia Mono / Consolas (Windows system)     | SIL Open Font License 1.1 | ⬜ not bundled |

**Current reality:** the CSS requests Inter and JetBrains Mono, but neither is
bundled or self-hosted, so the application currently renders in the Windows
system fallbacks. This looks correct on the target machine but is **not
deterministic across systems**.

Both intended fonts are SIL OFL 1.1, which permits bundling and redistribution.
Bundling them (with their licence files placed in `src/assets/fonts/`) is tracked
in `ROADMAP.md`. No font is loaded from a remote CDN — the CSP forbids it, and
remote font loading would leak a request on every launch.

## Adding an asset

1. Verify the licence permits use **and redistribution in a desktop application**.
2. Prefer original or procedurally generated work.
3. Add a row to this file before the asset is committed.
4. Place licence text alongside the asset.
5. Never load an asset from a remote origin — the CSP forbids it and it would
   leak a network request.
